Jenis - jenis Desktop Environment 
            Di dalam artikel kali ini, saya bermaksud untuk memberi tahu para pembaca tentang jenis-jenis desktop environment (DE). Dalam desktop environment ini ada jenis-jenisnya seperti KDE, GNOME, dan XFCE, dll.


            Desktop Environment (DE) umumnya merujuk ke gaya grafis antarmuka pengguna (GUI=Graphical User Interface) yang mememperlihatkan kiasan desktop di layar komputer modern. Lingkungan desktop adalah alternatif yang paling populer sebagai pengganti antarmuka baris perintah command line interface (CLI) yang saat ini umumnya digunakan terbatas oleh para pakar komputer.
Berikut ini merupakan daftar beberapa komponen DE yang dimiliki :
- Window Manager, merupakan aplikasi DE yang bertugas untuk mengelola interaksi terhadap window.
- Display Manager, merupakan aplikasi untuk mengelola resolusi layar dan dukungan multimonitor.
- Taskbar, merupakan komponen untuk menampilkan daftar aplikasi yang sedang digunakan oleh user.
-  Konfigurasi DE, untuk menampilkan semua konfigurasi yang ada pada DE.
-  Theme Manager, merupakan aplikasi untuk memodifikasi style ( look and feel ) pada DE.
-  Aplikasi pendukung lainnya seperti File Manager, Text Editor, Image Viewer, Audio Player, Video Player, dan lainnya.
*KDE
            KDE adalah desktop environment dan platform pengembangan apilkasi yang dibangun dengan toolkit QT dari tolltech. KDE dapat ditemui pada berbagai sistem UNIX, termasuk Linux, BSD, Solaris.
            KDE juga tersedia untuk Mac OS X dengan bantuan lapisan XII dan untuk Microsoft Windows dengan bantuan cygwin.

*GNOME
             GNOME adalah lingkungan desktop dan antarmuka grafik pengguna yang berjalan diatas sistem operasi. GNOME secara keseluruhan terdiri dari perangkat lunak bebas dan gratis. GNOME merupakan proyek internasional untuk menciptakan kerangka aplikasi perangkat lunak untuk desktop dan juga untuk mengatur peluncuran, penanganan file dan manajemen tugas jendela ( Window ).

*XFCE
            XFCE merupakan perangkat lunak bebas desktop untuk UNIX dan platform mirip UNIX lainnya seperti Linux, Solaris, dan BSD.
